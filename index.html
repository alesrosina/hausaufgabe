Simple demo for navigation menu

I have used SASS and Grunt for easier creation of CSS, where I have used some helper tools:

  • Autoprefixer: Prefixes all css properties, that are not covered by default in given browsers
  • Cssnano: Minimizing CSS file (removing all comments, whitespaces, etc.)
  • SASS: CSS precompiler.
  • Normalize.css: Used for normalizing all DOM elements for various browsers.
  • include-media: SCSS mixins for media queries.

For mobile I have chosen a solution, where a user would have a back button and current step. It does indicate which step user is currently visiting and allows a user to go back, but a downside of this approach is, that a user does not see how many steps are in front of him.
This might be solved by using "accordion style" (see image below) of navigation, where a user would see how many steps there are, and could also navigate around steps.
Both solutions works on different use cases, so it really depends on actual use case of this example.
I have also choose a solution with back button due to time constraint, since accordion solution would require a bit more brainstorming on how to achieve it with not too much of messing with clean HTML.

See it directly in action by going here (html/step-navigation.html) or scroll down to see it included in iframes for different screen sizes.
